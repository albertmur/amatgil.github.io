Bogomult

El revolucionari nou algorisme de multiplicació

En Rust:

fn bogomult(x: u64, y: u64) -> u64 { if x == 0 { return 0 } if x == 1 { return y } let mut n = 0; while n < bogomult(x - 1, y) + y { n += 1 } n }

En C++:

uint64_t bogomult(uint64_t x, uint64_t y){ if (x == 0) { return 0 } if (x == 1) { return y } uint64_t n = 0; while(n < bogomult(x - 1, y) + y) { n += 1 } return n; }

Explicació de l'algorisme

Bogomult rep dos nombres naturals \((x,y)\) i retorna el seu producte \((x \cdot y)\). A continuació s'explicarà pas per pas el seu funcionament.

Cas base
Si \( x = 0 \), la funció retorna 0: qualsevol nombre multiplicat per 0 és 0.
Si \( x = 1 \), la funció retorna y: qualsevol nombre multiplicat per 1 és aquell nombre.

Cas recursiu
Si \( x \) és major que 1, la funció fa la crida recursiva i comença el seu descens a l'infern. Es calcula el producte de \( x \cdot y \) sumant \( y \) \( x \)-vegades:
  • Primer es crida recursivament a \( bogomult(x-1,y) \), el que implica que s'està calculant \( y /cdot (x-1) \).
  • Després, s'afegeix \( y \) al que ens retorna la funció.
  • A continuació, el bucle while incrementa n des de 0 fins a arribar al valor \( bogomult(x - 1, y) + y \). Meravellós!

Exemple d'execució

  1. \( bogomult(3, 2) \) crida a \( bogomult(2, 2) \)
  2. \( bogomult(2, 2) \) crida a \( bogomult(1, 2) \)
  3. \( bogomult(1, 2) \) retorna 2 perque \( x = 1 \)
  4. \( bogomult(2, 2) \) inicia un bucle \( while \) que incrementa \( n \) des de 0 fins arribar a \( bogomult(1, 2) + 2\), és a dir, \( 2 + 2 = 4 \), i retorna \( 4 \).
  5. Finalment, \( bogomult(3, 2)\) comença un bucle while que incrementa \( n \) fins arribar a \( bogomult(2, 2) + 2\), és a dir, \( 4 + 2 = 6\), i retorna \( 6 \).
